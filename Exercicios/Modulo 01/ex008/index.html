Nesta frase temos esta palavra em destaque: "Teste 1,2,3 ... usando a tag STRONG(semântica).

Itálico / Enfase

Nesta frase, temos um termo em itálico usando a tag I(não semântico).

Nesta frase, temos um termo em ênfase usando a tag EM(semântica).

Texto marcado

É possivel criar um texto marcado usando a tag MARK.

Texto grande e pequeno

Aqui temos uma demonstração de um texto grande e um texto pequeno usando a teg BIG(Obsoleto) e SMALL.

Texto deletado

Podemos marcar um texto excluido para indicar que ele deve ser lido, mas não considerado usando a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois usando a tag INS.

Existe tambem o sublinhado com a tag U (não semântica).
